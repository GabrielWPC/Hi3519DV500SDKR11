执行步骤:
1.执行./sample_quick_start 0 0 0 0 0 0 0 0 0
2.收敛完成后退出用例，打印曝光参数
3.使用曝光参数重新执行./sample_quick_start
4.观察收敛效果

注意事项:
1.增益和曝光量需要对应，否则会影响收敛的速度
2.曝光量和环境要对应，通路起来就是收敛ok的状态
